Smartphone Market Analysis Dashboard

Power BI Data Analytics Project

Project Overview

This project analyzes smartphone specifications including price, RAM capacity, battery capacity, rear camera resolution, and screen resolution. The dataset was cleaned using Power Query and visualized using Power BI dashboards.

Tools Used

  • Power BI
  • Power Query
  • Data Visualization
  • Data Cleaning

Key Insights

  • Higher RAM capacity smartphones generally have higher prices.
  • Battery capacity varies significantly across smartphone brands.
  • Rear camera resolution comparison highlights top brands.
  • Screen resolution differences influence device pricing.
