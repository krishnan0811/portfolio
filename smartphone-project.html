Smartphone Market Analysis Dashboard

Power BI Data Analytics Project

Project Overview

This project analyzes smartphone specifications including price, RAM capacity, battery capacity, rear camera resolution, and screen resolution. The dataset was cleaned using Power Query and visualized using interactive Power BI dashboards.

Key Insights

  • Higher RAM → Higher price
  • Battery varies across brands
  • Camera comparison shows top brands
  • Screen resolution impacts pricing

Dashboard Preview

This dashboard analyzes smartphone market data by showing key KPIs, OS distribution, brand-wise model count, and price comparison. It helps identify dominant brands, pricing trends, and overall market patterns using interactive filters.

This dashboard analyzes smartphone performance data by visualizing key metrics such as RAM capacity, processor distribution, battery performance, and brand-wise comparisons. It helps identify top-performing brands, hardware trends, and overall smartphone market patterns using interactive visuals.
