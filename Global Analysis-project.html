Global University Analysis Dashboard

Python + Streamlit Data Analytics Project

Project Overview

This project analyzes global university rankings using Python and Streamlit. It provides interactive visualizations such as top universities, student distribution, university types, and location insights with dynamic filtering.

Tools Used

  • Streamlit

Key Insights

  • Top-ranked universities dominate global rankings
  • Most universities have moderate student populations
  • Private and public institutions are almost equally distributed
  • Location plays a key role in university concentration
